SimplyE Web Catalog

The SimplyE Platforms Web Catalog is an Open Publications Distribution System (OPDS) client web application which allows you to deploy a simple unified ebook catalog on your library's website. It is the same union catalog on the SimplyE mobile app. This allows you to create a dedicated, browse and searchable catalog of all you ebooks.

Live Example: The SimplyE Collection

The catalog below is a live example of the SimplyE Collection added to a web page.

Application
  • Catalog Integration
  • Online EBook Discovery
Bennefits

As opposed to linking out to separate vendor hosted web sites, catalogs and interfaces which only contain those book held by that vendor, ebook users see your entire ebook collection. You can also apply your own style sheet to the app so that it reflects your library's online look and feel.

Features
  • Borrow, reserve or download books with your library card.
  • Open non-DRM EPUBS and read in the browser
  • Use MARC 856 "Electronic Digital Resource Record" to link OPDS Web Catalog Book bibpage to your ILS catalog
  • Use the OPDS Book Entry url as a share link that can be forwarded to OPDS Ebook Readers (like SimplyE) to open the app to that particular book

How to implement into the MARC record

Example 1: Linking ILS catalog record to local eBook resource catalog

You can integrate SimplyE into your libaries catalog and web site by deploying an instance of the SimplyE Web Catalog on your web server. You can then link to it as a dedicated eBook catalog in your site navigation structure or simply insert it within your web site as an iframe. To continue the integration into your ILS catalog would simply take the website URLS as use them as 856 Electronic resource files to link out to from your catalog to your normailized ebook web bib page as you already do with Overdrive, B&T, Bibliotheca or other ebook content provider.

Example: NYPL book "Unbroken" Web link to place in MARC feild

856 40$u http://opds-browser-demo.herokuapp.com/collection/https%3A%2F%2Fcirculation.librarysimplified.org%2Fgroups%2F/book/https%3A%2F%2Fcirculation.librarysimplified.org%2Fworks%2FOverdrive%2FOverdrive%2520ID%2F7bf6eaad-0cfb-4c83-97c4-88db64d89fa9/

Example 2: Linking ILS catalog record to SimplyE OPDS Catalog resource locator

To make the link shareable so that users can open the SimplyE Mobile app to a partiucalar catalog page you can use the OPDS link as well in the electronic resource feild. You can also use this for other promotional purposes of a particular book. The users can follow the link and envoke the SimplyE App to open to a paricualr book in its catalog.

Example: NYPL book "Unbroken" OPDS link to place in MARC feild

856 42$uopds:https://circulation.librarysimplified.org/works/Overdrive/Overdrive%20ID/7bf6eaad-0cfb-4c83-97c4-88db64d89fa9

Using your web catalog in conjunction with book promotions

Example: QR codes for New Arrivals

The same MARC Resource Locator record URLs can be used in conjunction with QR codes. QR codes can be generated to represent a URL. These QR Codes can then be used in print materials such as posters or brochures highlting Staff Picks for books you want to promote. Users can then scan the associated QR code with a title to open their web browser or app to the Books catalog page.

QR Code for The NYPL's copy of As Kingfishers Catch Fire

The QR Code to the left represents the following URL: http://opds-browser-demo.herokuapp.com/collection/https%3A%2F%2Fcirculation.librarysimplified.org%2Fsearch%2F%3Fq%3DAs%2520Kingfishers%2520catch%2520fire/book/https%3A%2F%2Fcirculation.librarysimplified.org%2Fworks%2FOverdrive%2FOverdrive%2520ID%2F23e94ca8-acac-473b-a5ab-215d2264b98d/
Simply click the URL above and a web catalog will open on the bib page for As King Fishers Catch Fire.
Now download a QR Code Reader App onto you smartphone or tablet such as QR Reader for iPhone and then scan the QR Code on the left.
Your phone web browser should open to the same catalog page for As King Fishers Catch Fire.
