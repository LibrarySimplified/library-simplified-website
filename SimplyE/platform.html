SimplyE

The Library E-Reading Platform

Library Users

Web Catalog

The SimplyE Web Catalog provides a highly customizable open source OPAC for ebooks collectionthat libaries can use to provide library user the unified view of the libraries ebook collection. Users can browse the collection by genre, subject, audience or language classification or search for specific titles. Users can login and borrow books or view their e-bookshelf.

  • Works in IE, Safari, Chrome and Firefox
  • Works in Mobile Browsers
  • Users can borrow, place holds, return books
  • Users can login with library card and mange their E-bookshelf

Web Reader

The Web reader allows those who like to read on the web via their laptop, desktop or mobile device to easily read open acces materials and public domain titles online.

  • Use indepently with other content solutions
  • Deploy as part of the SimplyE Web Catalog

Try it at SimplyE.net

SimplyE Mobile

The SimplyE mobile application provides a native mobile interface for finding, borrowing and reading content on a variety of compatible platforms and devices such as Android, ChromeOS and Kindle Fire (if sideloaded). Users can pick their library if available or simplye read from the five thousands works available for free without a library card. The application works with most Library Ebook providers.

Works with

  • Android 4.2, 5, 6, 7, 8
  • iOS vers 8,9,10, 11
  • Overdrive, Bibliotheca Cloud Library, Access 360, Oddilo, Bioblioboard, Plympton Books
  • Download free content from Project Guttenberg, unGlue.it
  • Comes with free collection of 5000 works in 5 languages regardless of library affiliation

Users can

  • Browse and Search for titles across entire library E-Book collection
  • Login with only your library card
  • Sign up for library card

Circulation Manager

SimplyE Circulation Managers create the magic behind SimplyE. The circulation managers take adavnatage of cloud service providers architectures, and services to create a highly affordable, performant and scalable ebooks service.

  • Language: Python
  • Framework: Flask
  • Database: PostGresSQL
  • Repository and Issue Tracking: GitHub

Metadata Wrangler

The Metadata Wrangler is a separate service applications that helps normailize and enhance catalog metadata of service providers.

  • Language: Python
  • Framework: Flask
  • Database: PostGresSQL
  • Repository and Issue Tracking: GitHub

Analytics

Currently the application integrates with Gogole Analytics through a proxy that aggregates events and usage so as to annonymizes user data. Addtinally, the Circulation Manager Admin provides summary statistics of the collection via a dashboard directly from the Circulation Manager database.

  • Framework: ReactJS
  • Language: JavaScript
