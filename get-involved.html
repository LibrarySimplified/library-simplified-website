For Library Patrons

SimplyE iOS

If you are a New York Public Library patron, you can use the SimplyE app to borrow e-books from your iPhone, iPad, or iPod Touch. You will need a valid library card and PIN, and iOS 8.0 or later on your device. Take a tour of the iOS application.

SimplyE Android

If you are a New York Public Library patron, you can use the SimplyE app to borrow e-books from your Android tablet or smartphone. You will need a valid library card and PIN, and Android device. Take a tour of the Android application.

For Developers

Library Simplified Code Repository

The Library Simplified application and source code are available on GitHub. GitHub is a web-based Git repository hosting service, which offers all of the distributed revision control and source code management functionality of Git.

  • The repositories are located at https://github.com/NYPL-Simplfied.
  • The project has a general wiki for the project. It is can be found at https://github.com/NYPL-Simplified/Simplified/wiki.
