Common Questions. Answered in Plain English.

  • Is Library Simplified an App?

    Library implified is a collection of software including a mobile client applciation (aka an App) as well as related server software and code libraries that organizations may use to deliver digital content to their patrons for expirmentation or daily services. It is designed to provide a unique user experience for the lending of ebooks and otehr digital conetent. It is intended to work and integrate with technologies and systems common to libraries such as an Itegrated Library System/Library Management Systems (ILS/LMS), Online Public Acces Catalogue (OPAC) or Single Sign On (SSO) credential management system.

  • Do you offer any Support?

    The software is distributed for free with no support, warranty or guarantees.

  • Do you offer any Support?

    The software is distributed for free with no support, warranty or guarantees.

  • Can I participate in its development?

    We would love your help! Be reminded that all code extensions and contributions should be provided back to the larger communicty license free. By doing this, as a communicty we strengthen the solution and the community of libraries seeking to further the desstribution of knowledge and principles of open access. Contact us.

  • What about Updates?

    We will post updates and contributions to the code repository as needed. We are still working out how this comunity may be maintained so any suggestions would be greatly welcomed.

  • Do you offer any Customization?

    We provide the application source code so it may be customized for an individual organizations use. However, we ask that enhancements, bug fixes, or extensions be contributed back to the commuity so that other lirbaries, like yours, can bennefit from your contribution.
